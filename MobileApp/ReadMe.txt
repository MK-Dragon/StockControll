To Run the App Locally:

1) Install Python and all packages needed to run the app:
    - Python 3.10 or higher
    - Kivy
    - KivyMD 1.1.1 or 1.2.0
    - requests
    - pycryptodome
    - logging
2) BackEnd Component Locally (see ReadMe.txt in BackEnd Folder!)
3) Run the main.py Python file


To Run App on LAN

1) Install Python and all packages needed to run the app:
    - Python 3.10 or higher
    - Kivy
    - KivyMD 1.1.1 or 1.2.0
    - requests
    - pycryptodome
    - logging
2) BackEnd Component on LAN (see ReadMe.txt in BackEnd Folder!)
3) Run the main.py Python file
4) In the Login Screen Replace the IP '127.0.0.1' with the IP of your server
